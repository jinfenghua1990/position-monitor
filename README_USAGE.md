# position-monitor 使用流程（v2.1）

本系统的目标不是预测某一只股票必涨，而是：

- 建立交易纪律
- 建立交易记忆
- 建立长期统计优势
- 形成 AI 辅助交易系统

---

# 每日核心流程

建议固定：

```text
盘前 → 盘中 → 盘后
```

进行数据回传与分析。

---

# 1. 新建交易记录

当你准备：

- 买入
- 观察
- 加仓
- 卖出
- 放弃

时，记录交易。

推荐输入格式：

```text
记录交易：
股票：300XXX
名称：示例科技
动作：买入
评分：83
仓位：20%
止损：-4%
理由：
- 二波回踩
- 放量
- 站上5日线
- 板块热度高
预期：
3日内冲前高
```

---

# 2. 每日持仓回传

推荐方式：

- 同花顺复制文本
- OCR识别结果
- 手工整理文本

推荐格式：

```text
300001 示例科技 成本10.00 现价11.80 盈亏18%
600001 新能源A 成本20.00 现价18.80 盈亏-6%
```

系统会自动：

- 更新持仓
- 风控分析
- 二波分析
- 更新日报

---

# 3. 回写交易结果（非常重要）

3日或5日后，需要回写结果。

推荐格式：

```text
回写结果：
股票：300XXX
买入日期：2026-05-09
1日结果：+2%
3日结果：+5%
5日结果：-1%
最大盈利：+8%
最大回撤：-4%
退出原因：止盈
是否按计划：是
失败原因：
- 追高
- 板块退潮
复盘：
买点没问题，但卖点过早。
```

系统会自动统计：

- 不同评分区间胜率
- 平均收益
- 最大回撤
- 失败模式
- 纪律执行率

---

# 4. 二波观察池

系统会自动跟踪：

- 曾经盈利卖出的股票
- 回调后的二波机会
- 放量重新启动
- 接近前高

目标不是追涨，而是：

```text
识别你真正有优势的二波结构。
```

---

# 5. AI 学习方向

系统未来学习的重点不是：

```text
预测股票涨跌
```

而是：

- 哪种结构你最容易赚钱
- 哪种行情你容易亏损
- 哪种评分区间最稳定
- 哪种失败最常见
- 你的纪律问题
- 你的情绪问题

# 7. 最重要的原则

不要追求：

```text
95分神票
```

真正重要的是：

```text
长期正期望 + 风险控制 + 纪律执行
```

这是整个 AI 交易系统的核心。
